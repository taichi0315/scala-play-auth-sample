# ユーザー認証サンプルプロジェクト
- 「Scalaの文法・標準ライブラリは勉強したけど、どう活かせばいいかわからない。。」という方向けに作成したサンプルプロジェクトです。
- 質問は[TwitterのDM](https://twitter.com/nepp_yaga)まで

# アプリ立ち上げ

## DB立ち上げ
```
docker-compose up
```

## DBログイン
```
id:   user
pass: password
```

## sbtインストール
```
brew install sbt
```

## sbt立ち上げ
```
sbt run
```
http://localhost:9000
にアクセス
